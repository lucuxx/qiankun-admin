
### 文章完整数据格式:
```
article: Array<object> = [
    {
        title: string       - 文章标题;
        time: string        - 文章创建时间;
        updateTime: sring   - 文章最后更新时间;
        content: string     - 文章内容（html格式）
        contentMD: string,  - 文章内容（markdown格式）
        titleImg: string    - 文章图片
        tags: string        - 文章分类
        abstract: string    - 文章摘要
    }
]
```
